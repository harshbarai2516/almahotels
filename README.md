# 🏨 AlmaHotels

A modern hotel discovery web application built with React, Vite, and TailwindCSS.
Users can explore hotels, view detailed information, save favourites, and enjoy a smooth responsive UI.

---

## 🚀 Live Demo

👉 https://almahotels.vercel.app/

---

## ✨ Features

* 🔍 Browse hotels by country
* 🧾 Detailed hotel pages with amenities
* ❤️ Add / Remove hotels from wishlist
* 💾 Persistent wishlist using localStorage
* 🌗 Dark mode toggle
* 📄 Blog page with pagination
* 🎠 Hotel carousel for featured listings
* 📱 Fully responsive design
* ⚡ Fast performance with Vite

---

## 🛠️ Tech Stack

* **Frontend:** React + Vite
* **Styling:** TailwindCSS
* **State Management:** Context API + Redux Toolkit
* **Routing:** React Router
* **API:** DummyJSON / LiteAPI (for hotel data)
* **Deployment:** Vercel
